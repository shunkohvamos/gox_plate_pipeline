# この研究の進むべき先

研究の全体像・参照研究・現行パイプラインを踏まえ、「どこに向かうか」を整理した。

---

## 1. 目指すゴール（科学的ストーリー）

### 1.1 一言で
**酵素の熱耐性を最大化する組成最適化ランダムコポリマー**を、Learn–Design–Build–Test とベイズ最適化で効率的に設計する。

### 1.2 差別化・メッセージ
- **Tamasi との対応**: REA ではなく **t50（半減期）と FoG** を一貫して定量し、初速〜熱失活の時系列から再現可能に算出。BO の提案理由・run_id/manifest で追跡可能。
- **RSC HTS**: 結合強度と残存活性は相関しないと報告されているため、「熱安定性そのもの」を時系列で測る本アプローチの意義が明確。
- **Nagao**: 同様の GPR+獲得関数だが、**三角組成空間（MPC/BMA/MTAC）** と **アンカー補正** でラウンド間を統合。論文水準の図（三角・2×2・Observed vs Predicted・履歴）をそろえている。
- **Panganiban**: 「表面パターンに合わせた統計的モノマー分布」という設計思想を共有。本プロジェクトは **熱ストレス後の活性時系列から t50/REA/FoG を数値化** し、BO で組成を最適化する点でストーリーがはっきりしている。

### 1.3 実験サイクルのゴール
- **5 反復程度の Learn–Design–Build–Test** を回し、各ラウンドで BO が提案した組成を合成・測定。
- 最終的に「**トップ組成の同定**」「**反復ごとの FoG/t50 改善**」「**外れ値は黙って捨てず理由を残す**」を論文・SI で示す。

---

## 2. 進むべき方向（優先順）

### 2.1 最優先：実験サイクルを回す
- **Build**: BO の提案（`next_experiment_top5`）に基づき、PMBTA 系の組成を合成。
- **Test**: 熱安定性アッセイ（既存プレート解析）で t50 / FoG を取得。
- **Learn**: 得られたデータを `fog_plate_aware` / `bo_learning` に追加し、次の BO を実行。
- これを **数ラウンド** 繰り返し、「ラウンドが進むほど FoG が改善する」軌跡を三角図・ランキング図で示す。

### 2.2 論文・SI に向けた図・ドキュメント（すでに実装済みの活用）
現行パイプラインで **すでに出力している** ものを論文・SI で使う：

| 成果物 | 用途 | 参照での対応 |
|--------|------|--------------|
| 三角図（Mean / Std / EI / UCB） | 組成空間での予測・不確実性・獲得関数 | Tamasi の組成+REA マップ、Nagao の等高線 |
| 2×2 サロゲートマップ（xy / BMA–MTAC） | 同上の別視点 | 同上 |
| **Observed vs Predicted** | モデル当てはまり・外れ値の議論 | Tamasi の外れ値報告・Nagao の信頼区間 |
| **三角図上のラウンド別サンプリング位置** | BO の探索履歴 | Tamasi の反復ごと候補 |
| **次実験候補の三角図上マーカー** | 次に測る組成の可視化 | Nagao の追加点矢印に相当 |
| t50 / FoG ランキング（全ラウンド） | ポリマー間比較・改善の可視化 | Tamasi の REA 分布・トップ組成 |
| per-polymer 時系列・QC レポート | 再現性・初速窓選択の根拠 | Methods・SI の追跡可能性 |

→ **「足りない図」より、既にある図を Methods とキャプションでどう説明するか** を決める段階に来ている。

### 2.3 あるとよい追加（論文・監査の完成度）
- **Methods 用の 1 ページ**: t50 の定義（y0/2 または REA50）、FoG の式、初速窓選択の原則、BO の設定（獲得関数・バッチ数・アンカー補正）を **1 箇所にまとめた文書**。設定や default から自動生成できると理想的。
- **Residual の三角図**: 学習点での「観測値 − 予測値」を組成空間にプロット（実装済みなら SI の補足に）。
- **ラウンド別 FoG/t50 ランキング図**: 現状は CSV のみ。必要なら「ラウンドごとの棒グラフ」を 1 枚追加。

### 2.4 オプション（ストーリーが求める場合）
- **SHAP 的な特徴重要度**: Tamasi はモノマー・DP の重要度を SHAP で示している。組成 3 成分なら簡易的な感度解析や部分依存でも「何が効いているか」を議論できる。
- **FoG vs t50 のトレードオフ**: 2 軸散布図や Pareto  front で、多目的として扱う場合に使用。
- **DLS・ゼータ電位**: 既にグループディスカッションで取得しているなら、Tamasi の DLS/QCM と同様に「凝集・吸着」の補足として SI に載せられる。

---

## 3. 中期的な広げ方（組成・酵素）

- **組成空間の一般化**: 最適な MPC/BMA/MTAC 比が固まってきたら、BMA を **EHMA, LMA, PMPTSSi** などに置き換えて「疎水モジュールの比較」に進む（250812 研究方針で言及）。
- **酵素の拡張**: 主軸は GOx。副に **Lip**、さらに **β-Gal** など他酵素でクロス評価すると、Tamasi の「酵素間クロス評価」に相当するストーリーになる。
- **生物物理の裏付け**: 論文で refolding や構造維持を強く主張するなら、Tamasi 同様に CD/SAXS/DLS を追加する選択肢がある（現時点では必須ではない）。

---

## 4. まとめ：次の一歩

1. **直近**: 次の BO 提案（`next_experiment_top5`）に基づき 1 ラウンド「Build–Test」を実行し、学習データを更新して再 BO。ラウンドが 2〜3 以上になると「履歴付き三角図・ランキング」の説得力が増す。
2. **並行**: Methods 用の 1 ページ（t50 / FoG / 初速窓 / BO 設定）を、設定やコードからまとめる。既存の `AGENTS.md` や `.cursor/rules` の定義と整合させる。
3. **論文執筆時**: 既存出力（三角図・Observed vs Predicted・サンプリング履歴・次候補マーカー・ランキング）をそのまま SI に並べ、キャプションと Methods で「再現可能・追跡可能」を明示する。

これにより、**参照論文（Tamasi, Nagao, RSC HTS）の水準に合わせた「熱耐性の定量・BO の追跡・図の論文品質」を満たしたまま、研究の進むべき先（実験サイクル完走 → 論文化）に進める**。
